## 💸 **PONTOS DE PERDA NA CADEIA DE VALOR**

### **ETAPA 1 - PERDAS NA MATÉRIA-PRIMA (15-20%)**

#### **Perdas Reais:**
```
Coleta: 2-3% (frutas não maduras/estragadas no campo)
Transporte: 3-5% (dano, temperatura inadequada)
Recebimento: 1-2% (rejeição qualidade)
Armazenamento: 4-6% (contaminação, deterioração)

Total: 10-16% de perda ANTES de processar
```

#### **Impacto Financeiro:**
```
Cenário: Fábrica de sucos (R$500M/ano matéria-prima)
Perda 12% = R$60 milhões jogados fora
Custo oportunidade: R$15 milhões (lucro perdido)
```

#### **Dor Humana:**
- **Fazendeiro:** "Meu trabalho perdido no transporte"
- **Motorista:** "Não consigo controlar temperatura"
- **Recebedor:** "Tenho que rejeitar carga boa"

## 🎯 **EVENTOS CRÍTICOS NA CADEIA DE VALOR**

### **EVENTO 1: FALHA DE CALIBRAÇÃO (IMPACTO CASCATA)**

#### **Cenário Detalhado:**
```
08:00 - Válvula de dosagem inicia com erro de 2%
08:00-12:00 - Cada lote produzido com 2% a mais de matéria-prima
12:00-16:00 - Erro cresce para 5% (desgaste equipamento)
16:00-22:00 - Erro atinge 8% (falha progressiva)

Impacto Direto:
- Matéria-prima extra: R$50.000
- Produto fora especificação: R$25.000
- Retrabalho: R$15.000
- Energia desperdiçada: R$8.000
- Total dia: R$98.000

Impacto Cascata:
- Cliente final recebe produto inconsistente
- Reclamações aumentam 15%
- Vendas caem 3% no mês seguinte
- Reputação marca afetada
```

#### **Como Waste Guardian Resolve:**
```
08:01 - Sensor detecta anomalia de calibração
08:02 - Alerta tablet operador
08:03 - IA sugere recalibração específica
08:05 - Operador executa ajuste
08:06 - Processo normalizado
Perda evitada: R$98.000 + impactos cascata
```

- Energia ociosa: R$8.000
- Peça emergencial: R$15.000
- Total: R$320.000

Impacto Cascata:
- Pedidos não entregues: R$150.000
- Multas contratuais: R$45.000
- Clientes insatisfeitos: 25% churn risco
- Perda receita: R$750.000/ano
```

#### **Como Waste Guardian Resolve:**
```
06:01 - IA detecta padrão de vibração anormal
06:02 - Alerta preditivo: "Rolamento em 70% desgaste"
06:03 - Sistema sugere manutenção preventiva

22:00 - Manutenção programada (sem impacto produção)
