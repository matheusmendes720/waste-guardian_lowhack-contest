## ⏱️ Estrutura do Pitch (3 Minutos)

```
0:00 - 0:30  │ PROBLEMA
             │ "As indústrias de alimentos perdem X% de matéria-prima..."

0:30 - 1:30  │ SOLUÇÃO + DEMO
             │ "Waste Guardian em ação..."
             │ Navegar: Visão Geral → Detalhe → Gerar Ação IA → Ações

1:30 - 2:10  │ COMO FUNCIONA (Mendix + GenAI)
             │ "Um microflow chama a API da OpenAI que gera..."

2:10 - 2:45  │ IMPACTO + MODELO DE NEGÓCIO
             │ "Cada 1% reduzido = R$X mil de economia..."
             │ "Modelo B2B SaaS por linha monitorada"

2:45 - 3:00  │ PRÓXIMOS PASSOS
             │ "Conectar a dados reais de MES/ERP... piloto em planta"
```

---

## 🎯 Princípios Chave para o Pitch

1. **Clareza** — Problema, solução e impacto em linguagem simples
2. **Demo Primeiro** — Mostrar funcionando vale mais do que explicar
3. **Mendix + GenAI** — Mencionar explicitamente as tecnologias obrigatórias
4. **Números** — Métricas de impacto, não apenas afirmações
5. **Alinhamento com ODS** — Conectar explicitamente às ODS 9 e 12

---

## 📹 Dicas de Gravação

- **Resolução:** Mínimo de 720p
- **Áudio:** Limpar o áudio, sem ruído de fundo
- **Iluminação:** Rosto bem iluminado, luz frontal
- **Enquadramento:** Demo do produto visível + rosto (ou apenas a demo)
- **Tempo:** Grave 10s a mais no início e no final para edição
- **Backup:** Grave duas versões como garantia
