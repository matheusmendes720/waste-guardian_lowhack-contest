## ⏱️ Pitch Structure (3 Minutes)

```
0:00 - 0:30  │ PROBLEMA
             │ "Indústrias de alimentos perdem X% de matéria-prima..."

0:30 - 1:30  │ SOLUÇÃO + DEMO
             │ "Waste Guardian em ação..."
             │ Navegar: Visão Geral → Detalhe → Gerar Ação IA → Ações

1:30 - 2:10  │ COMO FUNCIONA (Mendix + GenAI)
             │ "Um microflow chama a API OpenAI que gera..."

2:10 - 2:45  │ IMPACTO + MODELO DE NEGÓCIO
             │ "Cada 1% reduzido = R$X mil economia..."
             │ "Modelo B2B SaaS por linha monitorada"

2:45 - 3:00  │ PRÓXIMOS PASSOS
             │ "Conectar em dados reais MES/ERP... piloto em planta"
```

---

## 🎯 Key Principles for Pitch

1. **Clareza** — Problema, solução, impacto em linguagem simples
2. **Demo First** — Mostrar funcionando vale mais que explicar
3. **Mendix + GenAI** — Explicitamente mencionar as tecnologias obrigatórias
4. **Números** — Métricas de impacto, não apenas afirmações
5. **ODS Alignment** — Conectar explicitamente às ODS 9 e 12

---

## 📹 Recording Tips

- **Resolution:** 720p minimum
- **Audio:** Limpar áudio, sem ruído de fundo
- **Iluminação:** Rosto bem iluminado, luz frontal
- **Enquadramento:** Product demo visível + rosto (ou somente demo)
- **Tempo:** Grave 10s a mais no início e final para editar
- **Backup:** Grave duas versões como seguro

## 🔝 Common Jury Questions

| Question | What They Want |
|----------|----------------|
| "Como a GenAI agrega valor real?" | Demonstra uso efetivo, não cosmético |
| "Qual o modelo de negócio?" | Viabilidade comercial |
| "Como mede impacto?" | Métricas de sustentabilidade |
| "Próximos passos?" | Potencial de continuidade |
| "Por que Mendix?" | Domínio da plataforma |
