## 🗑️ Food Waste in Brazil: The Problem

### Scale of Waste

| Statistic                      | Value                       | Context              |
| ------------------------------ | --------------------------- | -------------------- |
| **Total Food Waste**     | 20-82.1 million tonnes/year | Wide estimate range  |
| **Global Ranking**       | #6 highest waste globally   | 20M+ tonnes          |
| **Per Capita Waste**     | 102 kg/person/year          | Above global average |
| **Economic Loss**        | R$ 61.3 billion/year        | Estimated            |
| **Environmental Impact** | 10% of global GHG emissions | From food waste      |

### Industry-Specific Waste

| Segment                   | Waste % | Primary Causes                      |
| ------------------------- | ------- | ----------------------------------- |
| **Meat Processing** | 8-12%   | Trimming, spoilage, quality rejects |
| **Dairy**           | 5-8%    | Temperature breaches, shelf life    |
| **Beverages**       | 3-5%    | Overfill, packaging defects         |
| **Bakery**          | 6-10%   | Staling, overproduction             |
| **Processed Foods** | 5-7%    | Setup waste, changeovers            |
